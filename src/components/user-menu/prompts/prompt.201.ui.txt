핵심요구사항) 스타일 적용 시 다음 조건을 준수할 것.
                - CSS Module만 사용할 것. (:global, :root, !important 금지)
                - position-absolute 사용 금지 (flexbox만 사용)
                - 추가 애니메이션 없이 기본 동작만 구현할 것
                - 모든 색상 값이 정확히 적용되었는지 확인
                - 박스 섀도우와 테두리 반경이 부드럽게 표현되는지 확인
                - 레이아웃이 중앙 정렬되고 gap이 적절하게 적용되는지 확인
                - 반응형 레이아웃 고려 (최대 너비 제한으로 큰 화면 대응)
